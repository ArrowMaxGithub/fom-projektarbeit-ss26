A Python implementation of the card game Durak.

## Project management

Dependencies are tracked and managed via UV:  
https://docs.astral.sh/uv/guides/projects/#project-structure

To add/remove a dependency:  
```uv add new_dependency_name```  
```uv remove new_dependency_name```  

To run a python script:  
```uv run path_to_script```  

UV will automatically sync the lockfile and enter a virtual environment before executing the script.
